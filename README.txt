Uso de la libreria "Beautiful-soup" para categorizar componentes de una web, y descargar a placer lo que queramos, iterando con facilidad gracias a bs4.
También he usado 'Requests' para descargar el completo de código HTML para despues procesar con bs4.
Por ultimo, he usado PANDAS para crear un Dataframe, y presentar con el la información procesada obtenida con bs4.
